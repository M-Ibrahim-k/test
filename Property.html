Project Name

Technologia Property Website Design (Interview Task)

Duration

2 Days

My Role

Solo UX Designer – I was responsible for analyzing multiple visual directions, studying competitor platforms, and creating a fully responsive high-fidelity property landing page based on real client content.

Project Goal

The task was to create a clean and modern real estate landing page using content from Flippmode.com and visual inspiration from Offmarket.com and selected Envato templates. The goal was to combine premium aesthetics with practical layout flow to create a trustworthy, visually rich experience that helps convert real estate leads.

Research

I carefully studied the following references provided by the client:

  • Flippmode.com – Used for actual listing content, property details, and structure
  • Offmarket.com – Inspired design direction: premium hero layout, large text, and smooth animations
  • RealtorPal & Hotelin – Reference for clean section layout, scrolling rhythm, and real estate trust elements
  • Hasseland – UI flow, typography scaling, and whitespace balance

I made a content-to-structure map to align the Flippmode content into a more engaging, persuasive layout—based on CRO principles and emotional design for high-value listings.

Intended Audience

The design targets real estate buyers and investors looking for off-market properties, luxury homes, or deals in competitive regions. These users:

  • Expect modern, high-credibility design
  • Prefer strong visuals and minimal clutter
  • Need fast, actionable layouts that encourage trust and inquiries

User Testing Results

I shared the page prototype with 3 non-designer peers and asked them what stood out and what felt missing. Feedback highlights:

  • “Feels premium, like a brand I’d trust to invest with.”
  • “Love how it shows just enough without overwhelming.”
  • “I would definitely scroll all the way through—it feels smooth.”

Based on this, I slightly adjusted section heights, button weight, and mobile responsiveness for CTA stickiness.

Conclusion

This was a fast-paced, visual-heavy project that allowed me to combine design analysis, CRO best practices, and rapid UI delivery under a client-style brief. It deepened my understanding of how structure, typography, and visual storytelling impact decision-making in real estate. The client’s direction was interpreted into a clean and premium property page design ready for development.
